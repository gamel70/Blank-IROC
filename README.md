# IROC — On-Call Directory (template)

A Vite + React PWA. Placeholder data only.

## Run locally
```
npm install
npm run dev
```

## Build
```
npm run build     # outputs dist/
npm run preview   # test the build
```

## Deploy
Push to GitHub, then import the repo at vercel.com. Vercel auto-detects Vite.
Every push to `main` redeploys automatically.

Edit placeholder data in `src/App.jsx` (HOSPITALS, ROLES, rosterFor).
